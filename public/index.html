Yelp Data Visualization

What is this page about?

Provides visuals of the Yelp Big Data for restaurant owners or anyone looking to start a new restaurant. We provide different data visualizations that will serve as a powerful tool in gaining restaurant insights, what is trending, which city likes what, neighbourhood competition and market trends of a particular location. These visualizations can also be used for doing data analysis and gaining insights that are not visible by naked eyes.

Ratings Analysis

This section helps understanding the popularity of a particular cuisine across different states. The Rating buckets are 1, 1.5, 2, 2.5, 3, 3.5, 4, 4.5, 5

Select a Cuisine and then select multiple States to know the number of restaurants in different Ratings buckets.

Cuisines Analysis

This section gives the visualization of different cuisines spread out across different cities of a state. The Bubbles represent the number of restaurant in that cuisine. This visualization also helps knowing what is the favourite cuisine of the selected State

Select a State and then select multiple Cities to see the Cuisine Distribution of multiple Cuisines

Statewise Analysis

In order to compare market trends across multiple States the below visualizations serves as a great tool. These visualization helps understand the cuisine popularity across different states.

Select multiple States and then select multiple Cuisines for comparision

Price Range Analysis

This section helps understand how are the restaurants of a State divided across the four price ranges $, $$, $$$ and $$$$. Also, the visualization helps further segregate the 4 buckets of price across different cuisines.

Select a State and select multiple Cities to see the Price Range Distribution across multiple Cuisines
